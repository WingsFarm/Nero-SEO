# SEO Writing Rules — NERO Blog

## Title (H1)
- Written in Hebrew
- Clear and direct — no clickbait
- Includes the primary keyword
- 50–65 characters if possible

## Meta Title
- SEO optimized
- Includes primary keyword
- Include NERO brand name when relevant

## Meta Description
- 140–160 characters
- Written in Hebrew
- States a clear benefit
- Encourages the click
- Natural language — no keyword stuffing

## URL Slug
- Written in **English**
- Lowercase, hyphen-separated only
- Includes the main keyword
- No unnecessary filler words
- Example: `/blog/minimalist-mens-bracelets` not `/blog/how-to-choose-the-best-minimalist-mens-bracelets-guide`

## Article Length
- Target: **900–1500 words**
- Deep enough to rank, structured for readability
- Do not pad artificially — if the topic requires less, prioritize clarity over length

## Keyword Usage
- Primary keyword in: H1, first 100 words, at least 2 H2 headings, meta description
- Keyword density: 1–2% (no stuffing)
- Use semantic variations and synonyms throughout

## Article Structure

### Introduction (Hook)
- 2–4 short paragraphs
- Define the problem or topic clearly
- Set expectation for the reader
- No fluff

### Main Body (H2 + H3)
- Use clear H2 sections
- Use H3 for sub-sections when needed
- Paragraphs: 2–4 lines max
- Use bullet points when they aid clarity
- Keep reading flow clean and structured

### Practical Value (Required)
Every article must include:
- Clear explanations
- Real style advice
- Actionable tips
- Examples: outfits, occasions, combinations

### Closing Section
- 2–3 paragraph summary
- Soft brand-aligned closing statement
- Optional subtle CTA (never aggressive)

## Internal Links
- Add 2–4 internal links per post
- Use natural, descriptive anchor text — never "click here"
- Link to NERO collections or related blog posts when relevant

## Images
- Every post needs at least one image
- Save images inside the post folder: `context/posts/{slug}/images/`
- Do NOT use the top-level `context/images/` directory
- Alt text: describe the image naturally, include a keyword where it fits
- Compress images before uploading

## Formatting
- Clean Markdown or clean HTML
- No emojis
- **Bold** only for key concepts — do not overuse
- No large unbroken text blocks

## Authority & Credibility
- Reference material facts when relevant (e.g., 316L stainless steel properties)
- Explain benefits clearly and confidently
- Avoid unverified claims

## Final Output Format
Each completed blog post must include:
1. Title (H1)
2. Meta Title
3. Meta Description
4. URL Slug (English)
5. Full Article (structured in Hebrew)
6. Suggested Internal Links (if applicable)
7. Image Prompt (optional)
